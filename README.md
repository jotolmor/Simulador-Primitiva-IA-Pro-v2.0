# 🎰 Simulador Primitiva IA Pro

Este software utiliza Inteligencia Artificial y leyes de probabilidad para generar combinaciones de la Lotería Primitiva basadas en datos históricos reales (1986-2026).

## 🛠️ Instalación y Configuración

1. **Python**: Descárgalo en [python.org](https://www.python.org/). **Marca la casilla "Add Python to PATH"** al instalar.
2. **Archivos**: Coloca `super_creador.py` e `historico_limpio.csv` en la misma carpeta.
3. **Librerías**: El programa las instala solo. Si prefieres hacerlo manual, usa:
   ```powershell
   pip install pandas scikit-learn openpyxl
